# Diagramas

Esta pasta contém representações simplificadas da arquitetura da automação.

## Arquivos disponíveis

- [Arquitetura macro](arquitetura-macro.md): visão ponta a ponta do fluxo entre entrada, processamento, persistência e distribuição operacional.

## Objetivo

Os diagramas ajudam a entender a solução sem depender da abertura imediata dos workflows exportados.

Eles devem representar decisões arquiteturais e fluxo de informação, não detalhes internos de execução de nodes.
